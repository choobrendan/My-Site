SFPL 2023 Data Visualiser

A visualisation and prediction of SFPL (Simmed Fantasy Premier League) player stats as well as predicted points based on web-scraped data of simulated matches from Discord via the bot FPL.ai

The web-scraped data is scraped via Discord Chat Exporter, and data is then wrangled via code and statistics are put into various machine learning models to find and predict the points of each player of a particular gameweek.

This project assisted me winning the SFPL that year out of a total participant number of 68.
